# Purpose

This document evaluates the available tuition pricing models for SydTek Scholars and determines which model should be deployed during each phase of ecosystem growth.

---

# Model 1
## USD Reference Model

### Example

```text
AAI-301 Artificial Intelligence Foundations

Tuition: $99 USD
```

Students may only pay:

```text
Credit Card
Apple Pay
Shop Pay
PayPal
```

### Advantages

```text
Simple bookkeeping

Traditional accounting

Easy audits

No crypto complexity
```

### Disadvantages

```text
No $GCB utility

No treasury growth

No token adoption
```

### Recommended Use

```text
Not Recommended
```

# Model 2
## USD + Live $GCB Conversion

### Example

```text
Tuition: $99 USD

Current GCB Price:
$0.00000395

Amount Due:
25,063,291 GCB
```

### Formula

```text
USD Tuition ÷ GCB Price
```

### Advantages

```text
Stable revenue

Student flexibility

Treasury growth

Easy accounting

Supports automation
```

### Disadvantages

```text
Requires price feed

Requires calculator
```

### Recommended Use

```text
Primary Model
```

# Model 5
## Degree Treasury Model

Students lock tokens rather than spend them.

Example:

```text
Bachelor Program

500M GCB Treasury Deposit
```

Upon completion:

```text
Credential Issued

Portion Returned

Portion Retained
```

### Advantages

```text
Long-term holding

Treasury stability

Student commitment
```

### Complexity

```text
High
```

### Recommended Use

```text
Future Research
```
