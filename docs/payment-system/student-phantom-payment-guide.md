# Purpose

This guide explains how a student can pay SydTek Scholars tuition using $GCB through Phantom Wallet.

This is written for students, parents, homeschool families, and certificate learners.

---

# What You Need Before Paying

Before making a $GCB tuition payment, students should have:

```text
Phantom Wallet installed

$GCB available in Phantom

Shopify order number

Correct tuition amount

Official SydTek Scholars treasury wallet

Transaction verification form
```

---

# Important Rule

All SydTek Scholars tuition is priced in U.S. Dollars.

$GCB is accepted as a payment method.

```text
USD Tuition ÷ Live $GCB Price = Required $GCB Payment
```

---

# Example

```text
Course:
AAI-301 Artificial Intelligence Foundations

Tuition:
$99 USD

Live $GCB Price:
$0.00000395

Required Payment:
25,063,291 $GCB
```

---

# Step 1 — Select Course or Product

Go to the SydTek Scholars Shopify store.

Choose:

```text
Book

Course

Certificate

Curriculum

Microcredential

Membership
```

Add the item to cart.

---

# Step 2 — Select Pay with $GCB

At checkout, choose:

```text
Pay with $GCB via Phantom Wallet
```

---

# Step 3 — Review Payment Instructions

Shopify will display:

```text
Treasury Wallet Address

Required $GCB Amount

Verification Form Link

Order Number
```

Do not send payment until you confirm the wallet address.

---

# Step 4 — Open Phantom Wallet

Open Phantom.

Select:

```text
$GCB
```

Then choose:

```text
Send
```

---

# Step 5 — Enter Treasury Wallet

Paste the official SydTek Scholars treasury wallet:

```text
[INSERT TREASURY WALLET ADDRESS]
```

Double-check every character.

---

# Step 6 — Enter $GCB Amount

Enter the required amount shown at checkout.

Example:

```text
25,063,291 $GCB
```

---

# Step 7 — Confirm Transaction

Review:

```text
Token: $GCB

Amount: 25,063,291

Recipient: SydTek Scholars Treasury Wallet
```

Then confirm the transaction in Phantom.

---

# Step 8 — Copy Transaction Hash

After payment, copy the transaction hash from:

```text
Phantom transaction history

or

Solscan
```

---

# Step 9 — Submit Verification Form

Submit:

```text
Full Name

Email

Shopify Order Number

Course/Product

USD Tuition Amount

$GCB Amount Sent

Wallet Address

Transaction Hash
```

---

# Step 10 — Wait for Verification

SydTek Scholars will verify:

```text
Correct token

Correct amount

Correct wallet

Successful transaction

Blockchain timestamp
```

After verification, access will be activated.

---

# Payment Confirmation

Once verified, the student will receive:

```text
Course access

Download link

Certificate instructions

Credential registry update
```

depending on the product purchased.

---

# Student Warning

Do not send:

```text
SOL

USDC

Wrong token

Wrong amount

Payment to a wallet from social media comments

Payment before confirming the official treasury wallet
```

---

# Refund Note

Digital asset transactions may be irreversible.

SydTek Scholars will review payment issues manually, but blockchain transfers cannot always be reversed.
