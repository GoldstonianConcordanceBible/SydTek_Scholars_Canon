## Board of Trustees

**Minimum required for Cognia eligibility:** 3 members

Board responsibilities:
- Policy adoption and revision
- Financial oversight
- Accreditation progress review
- Curriculum standards approval
- Institutional mission alignment

Meeting cadence: Minimum quarterly
Documentation: `00-canon-governance/board/resolutions/`

---

## Version Control Policy

All policy documents are versioned (v1.0, v1.1, etc.).
Major updates require a Board Resolution filed in:
`00-canon-governance/board/resolutions/`

Changes are proposed via GitHub Issues and merged via Pull Requests.
Every merge is a public, timestamped record of institutional change.

---

## Privacy Boundary

This repository is public-safe. It contains:
- Institutional policy documents
- Curriculum frameworks
- Accreditation evidence structures
- Process documentation

It does NOT contain:
- Student personally identifiable information (PII)
- Student records or transcripts
- Financial account details
- Private board deliberations

Student records are maintained in a separate, private, FERPA-compliant system.

---

## Doctrinal Governance

All institutional decisions are evaluated against the Mirror → Water → Fire framework:

- **Mirror:** Does this decision reflect truth, accountability, and alignment with the GCB mission?
- **Water:** Does this decision refine and correct prior practice where needed?
- **Fire:** Does this decision execute toward legacy and 50-year institutional permanence?

---

## 50-Year Permanence Standard

SydTek Scholars is built to outlive its founder. Every system, document, and process is designed to operate without collapse in the founder's absence. The GitHub repository, DOI indexing, and public accreditation record are the institutional permanence infrastructure.

# Seat Time Model

## Per-Course Model

| Component | Count | Duration | Total |
|---|---|---|---|
| Video lessons | 20 | 10 min each | 200 minutes |
| Reflection per lesson | 20 | 5 min each | 100 minutes |
| Written assignment | 1 per unit (4 units) | 15 min each | 60 minutes |
| End-of-unit assessment | 4 | 20 min each | 80 minutes |
| **Total per course** | | | **~440 minutes (~7.3 hours)** |

| K–5 | 8 | ~5 hrs | ~40 hours |
| 6–8 | 10 | ~6 hrs | ~60 hours |

> Note: These are minimum documented instructional hours. Actual engagement typically exceeds these figures due to supplemental reading, parent-led discussion, and project work.

---

## Cognia Seat Time Alignment

Cognia does not mandate a specific seat time hour count for nontraditional online schools. Instead, Cognia evaluates:
1. Whether the instructional model is documented
2. Whether seat time is consistently tracked
3. Whether student achievement evidence exists

SydTek Scholars satisfies all three:
1. This document + `CURRICULUM_FRAMEWORK.md` provide full documentation
2. Parent dashboard progress logs + YouTube watch history provide tracking
3. Reflection assignments, written responses, and assessment records provide achievement evidence
