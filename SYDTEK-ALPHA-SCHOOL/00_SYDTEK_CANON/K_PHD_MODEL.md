# Purpose

This document defines the official K–PhD educational architecture for SydTek Alpha School.

It establishes how SydTek connects:

- Kindergarten;
- elementary education;
- middle school;
- high school;
- undergraduate education;
- graduate education;
- professional education;
- doctoral education.

The model is designed as one continuous learner pathway while preserving the developmental, academic, legal, and accreditation requirements appropriate to each educational level.
