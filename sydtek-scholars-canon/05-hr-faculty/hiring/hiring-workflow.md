# Hiring Workflow (v1.0)

## Purpose
Hire adults who can teach well, protect students, and follow integrity + privacy standards.

## Roles covered
- Teachers
- Academic Director
- Ops/Compliance Lead
- Safeguarding Lead

## Workflow
1. Define role + job description
2. Collect applications (private)
3. Screen for mission alignment + competence
4. Interview using scorecard
5. Reference check
6. Offer + onboarding checklist
7. Required training completion before student contact

## Documentation rules
- Applicant info is RESTRICTED (private storage).
- Public repo contains templates and workflow only.
