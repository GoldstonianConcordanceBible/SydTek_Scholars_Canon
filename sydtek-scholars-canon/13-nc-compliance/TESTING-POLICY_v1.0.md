# Testing Policy (v1.0)

## 1) Purpose
Establish consistent testing documentation practices and protect privacy.

## 2) Scope
Applies to any standardized testing used to demonstrate progress or meet program expectations.

## 3) Policy
- Testing choices, cadence, and documentation rules are standardized.
- Student-specific testing results are stored privately.
- Only blank templates and procedures are published publicly.

## 4) Procedure (public-safe)
1. Document the testing event using the approved template structure.
2. Store student-identifying results in private records storage.
3. Maintain record location references (no results published).

## 5) Records & Retention
- Testing records: retain per `RECORDS-RETENTION_v1.0.md`
