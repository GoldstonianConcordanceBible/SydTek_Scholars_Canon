# Immunization Records Handling (v1.0)

## 1) Purpose
Define privacy-safe handling and storage guidance for immunization records, if collected/maintained.

## 2) Scope
Applies to any immunization documents shared by families.

## 3) Policy
- Immunization records are treated as private sensitive data.
- They are never published in public repos.
- Access is role-based and logged.

## 4) Procedure (public-safe)
1. Store immunization documents only in the private records storage location.
2. Limit access to authorized roles only.
3. Log access and any transfers.
4. If a family revokes consent for a vendor tool, remove access accordingly.

## 5) Records & Retention
- Retain per `RECORDS-RETENTION_v1.0.md`
