# Audit Response Checklist (Public-Safe)

## 1) Intake
- [ ] Log the request date/time
- [ ] Identify requesting party and scope
- [ ] Confirm what is being requested (records/time window)

## 2) Prepare
- [ ] Identify record owners
- [ ] Pull only the minimum necessary records
- [ ] Review for privacy redaction needs
- [ ] Confirm access logs are complete

## 3) Respond
- [ ] Provide requested records via approved secure method
- [ ] Record what was provided and to whom
- [ ] Log any gaps found during retrieval

## 4) After-action
- [ ] Create corrective action items for gaps
- [ ] Publish a public-safe note (if appropriate) describing what changed (no private data)
