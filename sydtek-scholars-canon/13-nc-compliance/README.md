# NC Compliance Packet (Public-Safe)

This folder contains **public-safe** compliance documentation for operating a homeschool program in North Carolina.

## What is included
- Plain-language summary of recordkeeping expectations
- Public-safe templates (blank logs and forms)
- Retention standards and storage instructions
- Audit response checklist and memo templates

## What is NOT included (privacy boundary)
- Student-identifying attendance logs
- Student test results
- Immunization records

Those are stored privately, following the procedures described here.

## How to use
- Use templates for consistent recordkeeping
- Follow retention and storage rules
- Use audit response checklist when requested by oversight bodies
