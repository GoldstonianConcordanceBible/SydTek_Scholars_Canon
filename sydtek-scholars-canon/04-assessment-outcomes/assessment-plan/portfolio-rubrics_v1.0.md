# Portfolio Rubrics (v1.0)

## Purpose
Define what qualifies as a strong portfolio artifact.

## Artifact criteria (0–3 scale)
1. **Alignment to outcomes**
2. **Quality of work**
3. **Evidence of growth**
4. **Integrity (honest authorship + citations + disclosures where required)**

## Notes
- Portfolio selections should include a mix of skill areas.
- Any published sample must be REDACTED.
