# Covenant Pathway — Overview (v1.0)

## 1) Purpose
The Covenant Pathway is the K–12 curriculum spine that aligns:
- academic mastery
- formation and service
- Scripture literacy
- (where applicable) Hebrew Thread progression

## 2) Structure
- Grade bands: K–2, 3–5, 6–8, 9–12
- Each band includes:
  - ELA
  - Math
  - Science
  - History/Social Studies
  - Bible Literacy / Worldview Integration
  - Electives (as applicable)
  - Hebrew Thread (progressive)

## 3) Delivery model (summary)
- Weekly cadence published per grade band
- Unit structure: outcomes → lessons → checks → artifact(s) → rubric
