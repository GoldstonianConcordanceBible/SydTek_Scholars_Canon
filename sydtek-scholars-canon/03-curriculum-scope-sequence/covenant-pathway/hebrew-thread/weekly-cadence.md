# Weekly Cadence — Hebrew Thread (v1.0)

## Purpose
Define what “practice” means so learning is consistent.

## Minimum weekly cadence (by band)
### K–2 (light)
- 2–3 short sessions (5–10 minutes)
- focus: recognition + sound + small vocab

### 3–5
- 3 sessions (10–15 minutes)
- focus: vocab growth + short reading practice

### 6–8
- 3–4 sessions (15–20 minutes)
- focus: reading + grammar foundations

### 9–12
- 4–5 sessions (20–30 minutes)
- focus: coursework progression + proficiency goals

## Documentation
- teachers track completion privately
- portfolio artifacts selected quarterly
