# Hebrew Thread — Proficiency Rubrics (v1.0)

## Purpose
Provide measurable expectations by grade band.

## Rubric dimensions (used across bands)
1. Recognition (letters/words)
2. Pronunciation (age-appropriate)
3. Vocabulary mastery
4. Reading fluency (as applicable)
5. Comprehension (as applicable)

## Bands (starter levels)
### K–2
- Recognition: emerging → secure
- Vocabulary: small set mastered
- Comprehension: picture/meaning matching

### 3–5
- Recognition: secure
- Vocabulary: expanding set
- Reading: short guided reads
- Comprehension: simple meaning checks

### 6–8
- Grammar: foundational
- Reading: short passages with support
- Comprehension: summaries or guided translation

### 9–12
- Coursework: Hebrew I–III milestones
- Reading: increasing complexity
- Comprehension: structured interpretation with guidance

## Evidence artifacts (examples)
- flashcard mastery checks (private)
- short reading recordings (private)
- quiz results (private)
- portfolio samples (redacted if published)
