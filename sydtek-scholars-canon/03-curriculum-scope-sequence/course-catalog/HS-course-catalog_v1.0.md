# High School Course Catalog (v1.0)

## Purpose
Provide a clear list of offered HS courses and the documentation standard for each.

## Documentation standard (non-negotiable)
Each course must have:
- course description
- learning outcomes
- weekly cadence expectations
- assessment approach
- rubric links
- proof-of-learning artifacts
- integrity + AI rules

## Core course areas (starter)
- English Language Arts (English I–IV)
- Mathematics (Algebra I, Geometry, Algebra II, higher as offered)
- Science (Biology, Chemistry, Physics as offered)
- History/Social Studies (World History, US History, Civics/Econ)
- Bible Literacy (sequence as offered)
- Hebrew (sequence as offered)
- Technology / Digital Ethics (as offered)
- Electives (as offered)
- Capstone (as offered)
