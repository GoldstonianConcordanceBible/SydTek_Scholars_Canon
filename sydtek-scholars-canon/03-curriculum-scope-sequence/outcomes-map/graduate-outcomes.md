# Graduate Outcomes (v1.0)

## Purpose
Define what graduates should be able to do—academically and in character.

## Outcomes (starter set)
1. **Literacy mastery:** read deeply, write clearly, speak confidently.
2. **Quantitative reasoning:** solve problems, explain reasoning, use math responsibly.
3. **Scientific reasoning:** observe, test, interpret, and communicate findings.
4. **Historical literacy:** understand timelines, evidence, and civic responsibilities.
5. **Integrity:** honest work, accountable behavior, consistent discipline.
6. **Service:** contribute meaningfully to family/community.
7. **Worldview reasoning:** articulate beliefs respectfully and thoughtfully.
8. **Digital ethics:** use technology responsibly and safely.
9. **Hebrew proficiency (as applicable):** demonstrate progression consistent with the program’s offerings.

## Evidence
These outcomes are measured via:
- rubrics
- portfolios
- course completion artifacts
- capstone evidence (HS)
