# Worldview Integration Rubric (v1.0)

## Purpose
Make worldview integration measurable and non-performative.

## Rubric dimensions (0–3 scale)
### 1) Understanding
0: unclear  
1: basic recall  
2: accurate explanation  
3: deep understanding with nuance

### 2) Application
0: no application  
1: generic application  
2: specific application to scenario  
3: consistent application across contexts

### 3) Integrity
0: deceptive or inconsistent  
1: partially consistent  
2: consistent with evidence  
3: consistent + self-correcting when wrong

### 4) Respectful communication
0: disrespectful  
1: mixed  
2: respectful  
3: respectful + builds understanding

## Evidence artifacts
- reflection prompts
- essays/presentations
- service project reflections
- discussion notes (privacy-safe)
