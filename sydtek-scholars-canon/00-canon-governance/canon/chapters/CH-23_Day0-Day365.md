# CH-23 — Day 0 to Day 365 Roadmap
**Status:** Draft (v0.1)  
**Purpose:** Provide a one-year execution roadmap that ties documentation, operations, curriculum, and accreditation readiness into a single cadence.

---

## 1) Purpose
This roadmap exists to:
- prevent chaos and drift
- sequence what must be built first
- create public accountability milestones
- support a readiness-first accreditation posture

## 2) Scope
Applies to:
- documentation buildout
- implementation sequencing
- internal audits and reporting cadence
- benchmarking and improvement cycles

## 3) Roadmap phases (high-level)
### Phase A — Day 0–30 (Foundation)
- Canon chapters drafted
- Board governance templates live
- Policies pack skeleton created
- Compliance packet skeleton created

### Phase B — Day 31–90 (Implementation)
- Handbooks v1.0 published
- Curriculum scope/sequence published
- Core rubrics and assessment plan published
- First internal audit run

### Phase C — Day 91–180 (Stabilization)
- Monthly/quarterly reporting cadence established
- HR/training modules implemented
- Evidence index and owner matrix operational
- Benchmarking pack active

### Phase D — Day 181–365 (Readiness)
- Self-study narrative drafted
- Gap scans closed or actively managed
- Visiting team prep (if applicable)
- Public annual report published (public-safe)
