# Code of Conduct

We keep this repo constructive and audit-grade.

## Expected behavior
- Be respectful and specific
- Critique documents, not people
- Use evidence and clear reasoning

## Unacceptable behavior
- Harassment, threats, hate speech
- Posting private student information
- Spamming or disruptive behavior

## Reporting
Open an issue titled “Conduct Concern” (do not include private info). Maintainers will respond.
