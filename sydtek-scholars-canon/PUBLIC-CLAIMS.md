# Public Claims Policy

This file defines what can be claimed publicly in marketing, onboarding, and public communications.

## Allowed claims
- “We operate an audit-friendly governance and evidence library on GitHub.”
- “We maintain versioned policies, minutes, and resolutions.”
- “We provide faith-integrated education within our documented standards.”

## Restricted / must be qualified
- “Accredited” — only if formally granted by a named accreditor and verifiable
- “Certified” — only if issued by an external certifying body
- “Licensed” — only if a legal licensing regime applies and has been met
- Any claim implying government endorsement

## Prohibited claims
- Any claim that is false, unverifiable, or implies accreditation before it exists
- Any claim that reveals private student data

## Required disclaimer line (if discussing accreditation plans)
“We are in a documentation and readiness phase and will only claim accreditation if and when formally granted by an accrediting body.”
