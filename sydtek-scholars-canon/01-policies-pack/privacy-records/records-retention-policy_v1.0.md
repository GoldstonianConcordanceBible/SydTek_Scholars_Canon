# Records Retention Policy (v1.0)

## 1) Purpose
Define how long records are kept and how they are disposed of safely.

## 2) Scope
Applies to all records created by the program.

## 3) Policy
- Retention windows are defined by record type and classification.
- RESTRICTED records are stored privately and disposed of securely.
- Disposal actions are logged privately.

## 4) Starter retention table
| Record Type | Classification | Retention | Notes |
|---|---|---:|---|
| Policies/templates | PUBLIC | Ongoing | Keep version history |
| Board minutes/resolutions | PUBLIC | Ongoing | Public-safe versions |
| Attendance logs | RESTRICTED | TBD | Private storage only |
| Testing results | RESTRICTED | TBD | Private storage only |
| Transcripts | RESTRICTED | TBD | Private storage only |
| Incident reports | RESTRICTED | TBD | Private storage only |
| Access logs | RESTRICTED | TBD | Private storage only |

**TBD** values should be finalized by leadership and documented updates committed.

## 5) Secure disposal rule
When a record expires:
- delete/securely destroy
- log disposal in private system
