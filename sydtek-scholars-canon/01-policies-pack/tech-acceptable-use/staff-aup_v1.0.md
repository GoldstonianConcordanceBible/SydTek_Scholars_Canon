# Staff Acceptable Use Policy (AUP) (v1.0)

## 1) Purpose
Define staff/contractor expectations for safe tech use and data handling.

## 2) Scope
Applies to all adults operating program tools/accounts.

## 3) Policy
Staff will:
- use approved tools only
- protect accounts (strong passwords, no sharing)
- follow data classification rules
- avoid placing RESTRICTED data into unapproved tools
- follow incident response steps when issues occur
