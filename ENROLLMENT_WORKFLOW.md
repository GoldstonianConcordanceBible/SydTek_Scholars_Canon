PURPOSE

Establish the official enrollment workflow
for SydTek Scholars from application through
student dashboard activation, course placement,
Spanish placement, portfolio setup, and
accreditation-ready record creation.

⸻

ENROLLMENT WORKFLOW OVERVIEW

Application
Document Review
Grade Placement
Spanish Placement
Student ID Creation
Dashboard Activation
Course Assignment
Portfolio Folder Creation
Parent Orientation
Student Orientation
Enrollment Confirmation

⸻

STEP 1 — APPLICATION SUBMISSION

Parent or guardian submits enrollment application.
Required:
Student Information
Parent / Guardian Information
Grade Level Requested
Enrollment Tier
Prior Academic Records If Applicable
Technology Agreement
Handbook Acknowledgments

⸻

STEP 2 — DOCUMENT REVIEW

Registrar reviews:
Application
Student Information
Prior Records
Transfer Records
Portfolio Evidence
Parent Agreements
Technology Agreement

⸻

STEP 3 — COMPLIANCE REVIEW

Chief Compliance Officer reviews:
Required Documentation
Privacy Requirements
State Compliance Notes
Parent Acknowledgments
Student Handbook Acknowledgment
Data Privacy Acknowledgment

⸻

STEP 4 — GRADE PLACEMENT

Grade placement is determined through:
Prior Grade Completion
Academic Records
Portfolio Evidence
Parent Documentation
Faculty Review If Needed
Registrar Approval

⸻

STEP 5 — SPANISH PLACEMENT

Spanish placement is determined through:
Grade Level
Prior Spanish Study
Vocabulary Evidence
Oral Practice Evidence
Writing Sample
Transcripted Spanish Credit
Faculty Review If Needed

⸻

STEP 6 — STUDENT ID CREATION

Student ID Format:
STS-YYYY-LEVEL-0001

Example:

STS-2026-G01-0001
STS-2026-G09-0001

⸻

STEP 7 — DASHBOARD ACTIVATION

Parent Dashboard Activated
Student Dashboard Activated
Registrar Dashboard Updated
Faculty Review Dashboard Updated
Compliance Dashboard Updated

⸻

STEP 8 — COURSE ASSIGNMENT

Student is assigned:
Grade-Level Curriculum
Covenant Book
Reading / English Course
Writing Course
Mathematics Course
Science Course
Social Studies Course
Spanish Course
AI / Digital Literacy Course
Portfolio Workbook

⸻

STEP 9 — PORTFOLIO SETUP

Student portfolio folder is created.
Portfolio must track:
Reading Evidence
Writing Evidence
Math Evidence
Science Evidence
Social Studies Evidence
Covenant Evidence
Spanish Evidence
AI / Digital Literacy Evidence
Service Learning Evidence
Annual Review Evidence

⸻

STEP 10 — ORIENTATION

Parent Orientation Includes:
Dashboard Use
Attendance Verification
Portfolio Uploads
Spanish Practice Support
AI And Technology Rules
Annual Review Process
Student Orientation Includes:
Course Access
Portfolio Tasks
Spanish Practice
Digital Citizenship
Service Learning
Progress Tracker

⸻

STEP 11 — ENROLLMENT CONFIRMATION

Enrollment is confirmed when:
Application is complete.
Records are reviewed.
Grade placement is approved.
Student ID is created.
Dashboard access is active.
Courses are assigned.
Portfolio folder is created.
Parent orientation is complete.
