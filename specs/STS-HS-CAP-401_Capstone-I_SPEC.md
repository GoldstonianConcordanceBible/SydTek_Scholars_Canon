# STS-HS-CAP-401 — Capstone I (Spec) v1.0

**Grade Level:** 11–12 | **Credits:** 0.5–1.0 | **Prereq:** Instructor approval

## Description
Plan and begin a meaningful project demonstrating mastery and contribution.

## Outcomes
- Define a project question/problem clearly.
- Create a plan with milestones and evidence.
- Gather sources/data ethically.
- Produce initial deliverables and reflections.
- Present progress updates clearly.

## Assessments
Summative: project proposal + progress presentation  
Portfolio: proposal, milestone evidence, reflection

## Units
1) Proposal & Planning  
2) Research/Build Phase I  
3) Evidence & Feedback  
4) Progress Presentation
