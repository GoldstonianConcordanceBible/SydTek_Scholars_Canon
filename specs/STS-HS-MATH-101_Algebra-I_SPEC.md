# STS-HS-MATH-101 — Algebra I (Spec) v1.0

**Grade Level:** 9–10 | **Credits:** 1.0 | **Term:** Full year | **Prereq:** Pre-algebra readiness

## Description
Foundations of algebra: equations, functions, graphs, and modeling.

## Outcomes
- Solve linear equations/inequalities accurately.
- Interpret and graph functions.
- Model real situations with algebraic expressions.
- Explain reasoning in writing.
- Demonstrate mastery through cumulative checks.

## Assessments
Formative: weekly skill checks  
Summative: unit tests + performance tasks  
Portfolio: 2 modeling tasks + error-analysis reflection
## Units
1) Expressions & Equations  
2) Linear Functions  
3) Systems (intro)  
4) Modeling & Review
