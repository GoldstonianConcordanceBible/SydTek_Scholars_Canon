# STS-HS-CAP-402 — Capstone II (Spec) v1.0

**Grade Level:** 12 | **Credits:** 0.5–1.0 | **Prereq:** Capstone I (or equivalent)

## Description
Complete and present the capstone with a defensible evidence portfolio.

## Outcomes
- Complete deliverables to a defined standard.
- Document evidence and learning process clearly.
- Reflect on meaning → action (impact).
- Present outcomes with integrity and clarity.
- Package a final portfolio for review.

## Assessments
Summative: final product + final presentation + portfolio  
Portfolio: final evidence set + reflection

## Units
1) Build Phase II  
2) Testing/Revision  
3) Final Packaging  
4) Presentation & Reflection
