# STS-HS-HEB-101 — Hebrew I (Spec) v1.0

**Grade Level:** 9–12 | **Credits:** 1.0 | **Prereq:** None

## Description
Hebrew foundations: alphabet, pronunciation, basic vocabulary, and guided reading practice.

## Outcomes
- Recognize and write Hebrew letters accurately.
- Pronounce basic words and patterns.
- Master a foundational vocabulary set.
- Read short guided selections (level-appropriate).
- Demonstrate progress using proficiency checkpoints.

## Assessments
Summative: proficiency checkpoints + reading task  
Portfolio: vocabulary mastery log + reading samples (private)

## Units
1) Alphabet & Sounds  
2) Vocabulary Foundations  
3) Guided Reading Practice  
4) Proficiency Review
