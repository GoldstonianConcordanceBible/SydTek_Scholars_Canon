# STS-HS-BIB-102 — Biblical Literacy II (Spec) v1.0

**Grade Level:** 10–12 | **Credits:** 1.0 (or 0.5 if semester) | **Prereq:** Biblical Literacy I (recommended)

## Description
Deeper theme studies, ethics reasoning, and disciplined interpretation with structured writing.

## Outcomes
- Analyze themes across multiple books.
- Compare interpretations responsibly using evidence.
- Apply ethical reasoning to real-life scenarios respectfully.
- Create a structured study portfolio.
- Demonstrate consistent integrity in claims.

## Assessments
Summative: comparative theme study + applied ethics essay  
Portfolio: curated study set + reflection

## Units
1) Themes Across Scripture  
2) Ethics & Wisdom  
3) Interpretation with Evidence  
4) Portfolio + Presentation
